Affiliated with SAT 2018 at the Federated Logic Conference 2018   ·   July 9-12   ·   Oxford, UK

Execution Environment

The evaluation tracks will be run on the StarExec cluster. The maching specifications for the StarExec cluster compute nodes are available here.

Resource Limits

Main tracks: Per-instance time limit of 3600 seconds (1 hour), per-instance memory limit of 32 GB. Note that if the number of submissions is large we may have to use 1800 seconds as a per-instance time limit.

Incomplete track: Two distinct per-instance time limits: 60 seconds and 300 seconds. Per-instance memory limit of 32 GB.
